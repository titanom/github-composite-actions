# Contributing

## Prerequisites

1. Node.js v18
2. pnpm
3. [act](https://github.com/nektos/act#installation) to run GitHub Actions locally

## Formatting

```
pnpm format

# or to check if code is formatted correctly
pnpm format:check
```

## Test Actions Locally

```shell
act [<event>] [options]

# or if installed as GitHub CLI extension
gh act [<event>] [options]
```
